# Autoclicker
Simple mouse autoclicker

# Activation
Press <i><b>'*'</b></i> to activate <br />
Press <i><b>'Del'</b></i> to close <br />
Press <i><b>'Ctrl + Shift + e'</b></i> to change time interval <br />

<hr />

# The difference between exe files
Unlike <i><b>autoclicker.exe</b></i>, <i><b>autoclicker_invisible.exe</b></i> runs in the background and also reads clicks.
